project-root/

│   ├── gcs_uploader.py             ← GCS 업로드 함수 (CSV, JSON)
│   ├── naver_beauty_crawler.py     ← 상품 수집 클래스
│   ├── naver_review_crawler.py     ← 리뷰 수집 클래스

├── dags/                           ← Airflow DAG 파일 저장 디렉토리

│   └── naver_review_dag.py         ← 리뷰 수집 DAG
│
├── data/                           ← 로컬 임시 저장용 폴더 (테스트 시)
│   └── collected_data/
│
├── tests/                          ← 테스트 코드들 (pytest 등)
│
├── README.md
└── .env                            ← 환경변수 (GCP 인증, 버킷명 등)
